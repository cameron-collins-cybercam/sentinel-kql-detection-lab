# Microsoft Sentinel KQL Detection Lab

## Overview

This project demonstrates hands-on experience with Microsoft Sentinel SIEM, focusing on detection engineering using Kusto Query Language (KQL).

The lab simulates real-world security scenarios including brute force attacks, suspicious authentication activity, and anomaly detection.

## Objectives

* Develop KQL queries for threat detection
* Simulate SOC alert triage workflows
* Document incident response processes
* Build practical SIEM experience

## Technologies Used

* Microsoft Sentinel (SIEM)
* Azure Log Analytics
* KQL (Kusto Query Language)
* Windows Security Events

## Key Detections

* Brute Force Login Detection
* Suspicious Admin Login Activity
* Impossible Travel Detection

## Outcome

This lab demonstrates practical security monitoring, detection logic creation, and incident response workflows aligned with real-world SOC operations.
